# GitBook
[Gitbook](https//www.gitbook.com)

우측 물음표 클릭시 마크다운 편집모드로 전환가능


```
# Post List

> 'Post'목록을 보여줍니다.

**URL**
`/post/`

**Method**
`GET`

## Success Response

**HTTP Status code**
`200`

**content**
```
위와 같은 형태로  **설명**, **URL**, **Method**, **status code**, **content**(Json형태)로 작성한다

쓸 수 있는한 최대한 자세히 작성해준다.
